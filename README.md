# \# AWS Load Balanced Web Application

# 

# \## Project Overview

# 

# This project demonstrates the deployment of a highly available and scalable web application on AWS using an Application Load Balancer (ALB) and EC2 instances.

# 

# The application is deployed across multiple Availability Zones to ensure fault tolerance and high availability.
